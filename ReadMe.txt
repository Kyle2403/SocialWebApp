This is a web app mimicking a forum based on the Flask framework in python. It sets up a website locally hosted at port 443 where user can register,login, post content,....
Super admin username & password is "admin".

Install Flask: pip install Flask
Install NodeJs https://nodejs.org/en/download
Check to see if node is installed: node -v
Check to see if npm is installed: npm -v
Install CryptoJS: npm install crypto-js

Run: python web_app.py
